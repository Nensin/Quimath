# 🧪 Quimath

Quimath es una página web que ofrece calculadoras interactivas para realizar cálculos quimicos de forma rápida y sencilla.

## Funcionalidades:
  -Conversion entre moles y gramos
  -Molaridad
  -Disolución
  -Masa molar
  -Porciento de composción
  -Fórmula Empírica
## Tecnologías utilizadas 
  -HTML
  -CSS
  -JavaScript
