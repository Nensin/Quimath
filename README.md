# 🧪 Quimath

Quimath es una página web que ofrece calculadoras interactivas para realizar cálculos quimicos de forma rápida y sencilla.

## Funcionalidades:
  -Conversion entre moles y gramos-
  -Molaridad-
  -Disolución-
  -Masa molar-
  -Porciento de composción-
  -Fórmula Empírica-
## Tecnologías utilizadas 
  -HTML-
  -CSS-
  -JavaScript-
## Vista previa
https://nensin.github.io/Quimath/
## 📦 Cómo usar
1. Ingresa a la página
2. Selecciona la calculadora que deseas usar
3. Introduce los valores necesarios
4. Observa el resultado automáticamente
